# boss_pin
虽然Boss直聘上面可以跟 HR 直接沟通很实用，但是投递职位非常麻烦，需要一个一个的手动去点击，大多数沟通了还没有反应。
所以我今天就用 Selenium + Python 写了一个自动沟通的脚本。
写的时候发现，Boss直聘上面反 Selenium 措施也是很到位的。
